# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-03-31

### Added

- Initial release
- Static `PACKAGE_MAP` detection — maps 80+ npm packages to technology slugs instantly, no LLM or network call required
- Interactive grouped multi-select to pick skills by technology
- Curated skills group (`⭐ curated`) with high-value Next.js and web quality skills
- Support for 30+ coding agents (Claude Code, Cursor, GitHub Copilot, Windsurf, and more)
- Concurrent skill installation with animated spinners
